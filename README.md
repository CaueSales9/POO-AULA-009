Sistema de Carros (POO). Projeto pratico de POO em Python, demonstrando herança e polimorfismo.
Arquivos do Projeto:
Classes

- carro.py: Classe base com métodos para acelerar, freiar e exibir velocidade.
- carro_esportivo.py: Subclasse que adiciona o método turbo.
- carro_inteligente.py: Subclasse que adiciona o método de estacionar automaticamente.
- carro_corrida.py: Subclasse para carros de competição.


Execução:
python main.py
